# PLM产品生命周期管理系统 - 全息投影版

## 🌟 系统概述

PLM产品生命周期管理系统是一个采用全息投影视觉风格的现代化企业级应用，专为产品全生命周期管理而设计。系统融合了未来科技感的全息投影效果，提供直观、高效的产品管理体验。

### 核心特征
- **全息渐变图层**: 采用 #00e5ff→#0080ff 渐变色彩
- **半透明数据投影**: 使用 opacity 0.6 的透明效果
- **轻量扫描动效**: 数据加载时触发全息扫描（1秒时长，无循环）
- **科技蓝主色**: 以全息蓝为主要视觉元素

### 适用场景
- 工业互联网平台
- 设备全息监控系统  
- 高端医疗数字化平台
- 智能制造管理系统

## 🚀 快速开始

### 环境要求
- Node.js 16.0+
- npm 7.0+
- 现代浏览器（支持CSS3和ES6+）

### 安装步骤

1. **克隆项目**
```bash
git clone [项目地址]
cd 03-PLM产品生命周期管理系统
```

2. **安装依赖**
```bash
npm install
```

3. **启动开发服务器**
```bash
# 方式一：使用npm命令
npm run dev

# 方式二：使用启动脚本（Windows）
start-plm-server.bat
```

4. **访问系统**
```
http://localhost:3003
```

## 🎨 全息投影特效

### 视觉设计要点
- **配色方案**: 
  - 全息色: #00e5ff→#0080ff
  - 背景色: #0a1929
  - 投影透明度: opacity 0.6

- **动效设计**:
  - 数据加载扫描: 从左到右，1秒时长
  - 无循环动画，避免视觉疲劳
  - 轻量级动效，保证性能

- **控件设计**:
  - 核心数据模块使用全息效果
  - 操作区域保持扁平化设计
  - 平衡视觉效果与用户体验

### 特效功能
- **全息扫描**: 点击元素触发扫描动效
- **数据投影**: 悬浮显示关键信息
- **状态指示**: 实时状态的全息显示
- **粒子效果**: 增强科技感的粒子动画

- **UI组件**: Element Plus

- **状态管理**: Pinia

## 🔧 开发指南

### 开发环境设置
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 自定义全息效果
```javascript
// 触发扫描效果
window.triggerHolographicScan(element)

// 全局扫描
window.holographicDataLoad()

// 创建数据投影
window.createDataProjection(element, data)

// 脉冲效果
window.holographicPulse(element)
```

### 样式变量使用
```scss
// 使用全息色彩
.my-element {
  background: $holographic-gradient;
  border: $border-holographic;
  box-shadow: $shadow-holographic;
}

// 使用动画混合器
.my-card {
  @include holographic-card;
  @include scan-animation;
}
```

## 🐛 问题排查

### 常见问题
1. **端口占用**: 修改vite.config.js中的端口配置
2. **依赖安装失败**: 清除node_modules重新安装
3. **全息效果不显示**: 检查浏览器CSS3支持
4. **图表不显示**: 确认ECharts正确加载

### 调试模式
